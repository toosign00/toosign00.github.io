## 지킬 테마를 이용하여 Github 블로그를 만들어보자
jekyll-theme-chirpy를 이용해 GitHub 블로그를 만들어 보았다.<br>
블로그를 만들기 전에는 여러 가지 어려움이 많았다. Ruby 설치부터 터미널 설정까지, 이외에도 정말 많은 자잘한 오류들이 발생했다. 하루 종일 붙잡고 고생한 끝에 드디어 완성할 수 있었다.<br>
오늘은 간단하게만 작성하고, 추후 지킬 테마를 적용한 과정들을 추가해서 작성해 볼까 한다.

## 댓글 기능 추가
블로그를 만들고 또 뭐를 할 수 있나 찾아봤는데 댓글 기능을 추가할 수 있다더라.<br>
그래서 블로그의 댓글 플랫폼을 찾아보다 disqus를 찾게 되었다. 구글링해서 찾아보니 방법은 크게 어렵지 않아 금방 댓글 기능을 추가할 수 있었다.<br>
아.. 근데 적용시키고 보니 일단 안 예쁘다. 어디 광고 많은 웹사이트에 있을만한 댓글 창 같았다. (실제로 광고도 많이 붙고 무겁다고 하더라)<br>
그래서 다른 댓글 플랫폼을 찾아보다가 utterances를 발견했다. 일단 광고가 없고, 가볍다고 한다. 그리고 댓글 창이 예쁘다. 그래서 이걸로 갈아타려고 한다.
빠른 시일 내에 utterances로 바꿔서 적용시켜 봐야겠다.<br><br>
p.s 마크다운 파일 작성이 익숙하지가 않다. 지금 줄간격 부터 맘에 안드는게 한두가지가 아님. html이 더 편하다 ㅠ
